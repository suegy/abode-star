Abode-star

ABODE-star is an integrated development environment (IDE) for designing behaviour based agents.

ABODE-star is an integrated development environment (IDE) for Behavior Oriented Design (BOD).

It is now still in beta mode (despite it's numbering scheme) and allows for the creation of POSH plans. The plans are able to be used in either posh# or jyPOSH.

For further information on ABODE or BOD consult the project website: Joanna Bryson's ABODE page

We currently provide a couple of showcases of what has been done using ABODE-star and POSH.
